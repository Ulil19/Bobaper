Bobapeer
Bobapeer adalah aplikasi E-Commerce berbasis web yang dirancang untuk membantu pengguna menilai, dan berbagi rekomendasi minuman boba dari kedai Bobapeer. Aplikasi ini menghadirkan pengalaman berbagi ulasan yang mudah, cepat, dan interaktif.

Dengan antarmuka yang sederhana, Bobapeer mendorong kolaborasi antarpecinta boba untuk memberikan informasi yang akurat tentang rasa, harga, pelayanan, hingga suasana kedai Bobapeer.
